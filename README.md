## Usage
Retrieve your _application-key_ from Application Settings in LogFlake UI.
```python
from logflake import logflake

logger = logflake.LogFlake('application-key')
logger.send_log(logflake.LogLevels.DEBUG, None, 'Hello World')

logger.shutdown()
```
